ENGINE ERROR
An error has occurred while communicating with the active engine. This might
be a result of one or more of the followings:

  • A misconfiguration was made on the Engines page for the active engine
  • The engine has been shut down
  • The engine encountered an unexpected error
In the first case, you can correct the configuration on the Engines page. If the
issues is not resolved within 3 minutes you will get redirected to this error
page again.

In the latter case, please submit the engine logs with your error report.
